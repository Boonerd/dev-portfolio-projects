# LexFlow Legal Solutions

> **A premium, dark-mode-enabled, fully responsive landing page for a fictional tech-forward law firm.**  
> Built to showcase **React + Tailwind + TypeScript + Framer Motion** mastery.

---

## Live Demo
[https://lexflow-legal.vercel.app](https://lexflow-legal.vercel.app)

---

## Features
| Feature | Tech |
|-------|------|
| Dark/Light Mode | `useState` + CSS Variables |
| Mobile Hamburger Menu | React + Tailwind |
| Custom SVG Gavel Graphics | Inline SVG |
| Scroll Animations | Framer Motion |
| Responsive Grid & Typography | Tailwind Mobile-First |
| Premium Law Firm Design | Slate + Gold |
